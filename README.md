# MokiRuby

A ruby gem for interacting with the Moki API, as a part of Moki Total
Control and Moki Management.

## Installation

Add this line to your application's Gemfile:

    gem 'moki_ruby'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install moki_ruby
